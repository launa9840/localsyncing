## Step-by-Step Process

### 1. Enable Password Protection
```
User Action: Click Settings â†’ Toggle Password ON
System: Shows password dialog
User Action: Enter password "mySecurePass123"
System: Computes SHA-256 hash client-side
System: Sends hash to server
Server: Stores hash with clipboard data
System: Shows "Protected" badge
Result: Clipboard is now password-protected
```

### 2. Access from Another Device
```
Device 2: Opens LocalSync
System: Detects clipboard is locked
System: Shows "Clipboard Locked" screen
User Action: Clicks "Unlock" button
System: Shows unlock dialog
User Action: Enters password "mySecurePass123"
System: Computes SHA-256 hash client-side
System: Sends hash to server for verification
Server: Compares hashes
Server: Returns success if match
System: Grants access to clipboard
Result: User can now view and edit content
```

### 3. Disable Password Protection
```
User Action: Click Settings â†’ Toggle Password OFF
System: Shows password dialog
User Action: Enter current password
System: Computes hash and verifies
Server: Confirms password is correct
Server: Removes password protection
System: Removes "Protected" badge
Result: Clipboard is now accessible without password
```
